Easy:bit

Övning 3 - Ja eller nej

Intro

I denna övning kommer du lära dig om hur du kan starta en funktion med hjälp av knapptryck, hur en enkel sant/falskt funktion fungerar, samt hur du kan lägga till ljud till dina funktioner.

Steg 1

Hämta inputblocket on button A pressed från kategorin Input, hämta sedan de dubbelklamradrade if true then else ur kategorin Logic och placera det innaför klamrarna av on button A pressed.

Steg 2

Hämta blocket pick random true or false från kategorin Math och placera det ovanpå true i blocket if true then else. Detta betyder att när du trycker på A-knappen i din micro:bit så kommer den slupmässigt välja om den skall aktivera klammern true eller else i if true then else-blocket.

Steg 3

Hämta nu två show string-kodblock ur Basic och placer dem i true-klammern och i else-klammern. Bestäm värdet på dem båda. Förslagsvis "Ja" i true och "Nej" i else. Detta innebär att när du trycker på A knappen i din microbit så kommer den slumpa mellan att visa "Ja" eller "Nej". För att göra effekten av detta starkare kan du sedan hämta två av blocket play tone ur Music och placera vardera i true och else tillsammans med show string-blocken. Du kan sedan välja en ton som spelas upp i samband med att if-blocket ger ett utslag. Till exempel en ljus ton för "Ja" och en mörk ton för "Nej".

Behöver du mer hjälp?

Använd dig utav vår ordbok för mer information om de individuella kodblocken. Du kan hämta, eller bläddra i den genom att klicka på denna länk.
